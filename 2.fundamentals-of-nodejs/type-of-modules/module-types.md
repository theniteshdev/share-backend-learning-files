# types of module

- USER MODULES- user defined modules for their use
- NPM MODULES- moudles downloded from npmhjs.com
- CORE / NATIVE MODULES- inbuilt modules provided by node

core-modules example-

```javascript
import fs from "node:fs";
import dgram from "node:dgram";
import crypto from "node:crypto";
import http from "node:http";
import https from "node:https";
```

(note- :node is optional but it is <mark>best practice.</mark>)
